authorizing(method = nil, &block)

Define a new Authorization Block.

A sequence of Authorization Blocks compose the Authorization Chain. Each model AC is evaluated when requesting authorization permissions to each instance. See ActiveRecord::Authorization

    # Grants all permissions to self
    authorizing do |agent, permission|
      agent == self
