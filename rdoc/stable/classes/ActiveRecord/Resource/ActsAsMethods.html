acts_as_resource(options = {})

Provides an ActiveRecord model with Resource capabilities

Options:

  • :mime_types - array of Mime::Type supported by this Resource.
  • :has_media - this Content has attachment data. Supported plugins: AttachmentFu (:attachment_fu)
  • :param - used to find and build the URLs. Defaults to :id
  • :disposition - specifies whether the Resource will be shown inline or as attachment (see Rails send_file method). Defaults to :attachment
  • :per_page - number of Resources shown per page, using will_pagination plugin. Defaults to 9
  • :delegate_content_types - allow using ActiveRecord::Resource.class_supporting for finding the class most suitable for some media. This is useful if you have a generic model for Attachments, but specific files like images or audios should be managed by other classes. Defaults to false
