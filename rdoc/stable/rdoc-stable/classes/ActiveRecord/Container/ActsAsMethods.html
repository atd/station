acts_as_container(options = {})

Provides an ActiveRecord model with Container capabilities

Options:

contents:an Array of Contents that can be posted to this Container. Ex: [ :article, :image ]. Defaults to all available Content models.
sources:The container has remote sources. It will import Atom/RSS feeds as contents. See Source. Defaults to false
