acts_as_agent(options = {})

Provides an ActiveRecord model with Agent capabilities

Options

authentication:Array with Authentication methods supported for this Agent. Defaults to [ :login_and_password, :openid, :cookie_token ].
openid_server:Support for OpenID Server. Defaults to false
activation:Agent must verify email. Defaults to false
invite:Agent can be invited to application. Can be false. Defaults to :email
