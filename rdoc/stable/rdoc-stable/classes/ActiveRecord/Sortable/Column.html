data(helper, object)

Get data for this object based in :content parameter. There are two types of :content parameter:

Symbol:represents a method of the object, like object.name
Proc:more complicate data. Example: 
  :content => proc{ |helper, object|
    helper.link_to(object.container.name, helper.polymorhic_path(object.container))
  }
