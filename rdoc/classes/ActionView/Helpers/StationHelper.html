title(new_title = "", options = {})

Get title in this order:

  1. string argument
  2. @title instance variable
  3. Title based on variables set by the Controller
  4. controller.controller_name - current_site.name

Options:

append_site_name:Append the Site name to the title, ie, "Title - Example Site". Defaults to false
